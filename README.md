# 游戏人生 - 比比的28年游戏历程

一个基于 Nuxt 4 构建的个人游戏人生展示网站，记录了从 1998 年到 2026 年的游戏历程。

## 技术栈

- **Nuxt 4** - Vue 3 全栈框架
- **Vue 3** - 渐进式 JavaScript 框架
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Google Fonts** - Noto Sans SC + Oxanium 字体

## 特性

- 🎮 赛博朋克风格设计
- ✨ 霓虹灯文字效果
- 🌊 玻璃态卡片样式
- 📱 响应式布局
- 🎯 滚动进度条
- 🖱️ 鼠标跟随光效
- 🎨 渐变背景和动画效果

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器 (http://localhost:3004)
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 项目结构

```
.
├── app.vue              # 主应用组件
├── assets/

```

## 配置说明

### 开发服务器

- 端口: `3004`
- 局域网访问: 已启用 (`host: 0.0.0.0`)
- 可通过 `http://192.168.x.x:3004` 在局域网内访问

### SEO 配置

- 标题: "我不是在玩游戏 - 游戏人生"
- 描述: "比比的28年游戏人生 (1998-2026)"

## 内容章节

1. **序章：个人档案** - 基本信息和游戏生涯数据
2. **第一幕：童年与启蒙** (1998-2006) - 街机厅、小霸王、Game Boy
3. **第二幕：成长与探索** (2006-2019) - PSP、网游、3DS
4. **第三幕：硬核与沉淀** (2019-2026) - Steam、Switch、完美主义
5. **终章：游戏与人生** - 人生感悟
